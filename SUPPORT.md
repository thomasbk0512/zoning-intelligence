### Common Issues

#### No Results Found

- Verify APN is correct
- Check coordinates are valid
- Ensure property is in Austin, TX

#### Network Errors

- Check internet connection
- Try again after a few moments
- Check browser console for errors

#### Map Not Loading

- Check browser console for errors
- Ensure JavaScript is enabled
- Try refreshing the page

## Reporting Issues

### Before Reporting

1. **Check Documentation**: Review [USER_GUIDE.md](USER_GUIDE.md) and [DIAGNOSTICS.md](DIAGNOSTICS.md)
2. **Collect Diagnostics**: Access diagnostics via `?debug=1` and copy information
3. **Check Console**: Open browser DevTools and check for errors

### Issue Report Template

When reporting issues, please include:

1. **Description**: What you were doing and what happened
2. **Steps to Reproduce**: Detailed steps to reproduce the issue
3. **Expected Behavior**: What you expected to happen
4. **Actual Behavior**: What actually happened
5. **Diagnostics**: Copy diagnostics information (via `?debug=1`)
6. **Browser Information**: Browser version, operating system
7. **Console Errors**: Any errors from browser console

## Feature Requests

Feature requests are welcome! Please create a GitHub issue with:
- Description of the feature
- Use case or motivation
- Proposed implementation (if applicable)

## Accessibility Support

If you encounter accessibility barriers:
1. Check [ACCESSIBILITY_STATEMENT.md](ACCESSIBILITY_STATEMENT.md)
2. Report via GitHub Issues with "accessibility" label
3. Include assistive technology used and browser information

## Security Issues

**Do not** report security issues via public channels. See [SECURITY.md](SECURITY.md) for responsible disclosure.

## Response Times

- **Critical Issues**: Within 24 hours
- **General Issues**: Within 3-5 business days
- **Feature Requests**: Reviewed during planning cycles
